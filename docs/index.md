## GitHub Pages

This site is powered by GitHub Pages. To see the API documentation as a separate tab:

1. Go to repo **Settings → Pages**
2. Set **Source** to `main` branch, `/docs` folder
3. Click **Save**

Then visit `https://your-username.github.io/joy-pay/`
